```tsx
import { reactBigCalendarAdapter } from '@consta/react-big-calendar-adapter/ReactBigCalendarAdapter';
```

Этот адаптер стилизует календарь [react-big-calendar](https://jquense.github.io/react-big-calendar/examples/?path=/docs/about-big-calendar--page) для дизайн-системы Consta.
Кроме стилей, адаптер добавляет в календарь перевод на русский и возможность посмотреть события за квартал.
