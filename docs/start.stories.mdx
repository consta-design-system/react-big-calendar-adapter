# reactBigCalendarAdapter | Дизайн-система Consta

Адаптер для календаря из библиотеки [react-big-calendar](http://jquense.github.io/react-big-calendar/examples/index.html?path=/story/about-big-calendar--page). Стилизует календарь для [дизайн-системы Consta](https://consta.design/).

[Посмотреть на GitHub](https://github.com/consta-design-system/react-big-calendar-adapter)

#### Как создать стилизованный календарь

- Подключите адаптер [reactBigCalendarAdapter](?path=/docs/utils-useReactBigCalendarAdapter--playground) к календарю, чтобы добавить стили для некоторых свойств.

## Быстрый старт

#### Установите пакет и зависимости

`yarn add @consta/uikit @consta/react-big-calendar-adapter` или `npm install @consta/uikit @consta/react-big-calendar-adapter`

#### Подключите тему

Тему обязательно нужно подключать в корне приложения, так, чтобы все компоненты `@consta/react-big-calendar-adapter` были вложены в компонент `Theme`.

[Как подключить тему](https://consta-uikit.vercel.app/?path=/docs/thematization-how-to-connect-theme--page)
